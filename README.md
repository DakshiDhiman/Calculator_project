# Calculator_project

## Introduction
The Calculator App is a simple Android application designed to perform basic mathematical calculations. The purpose of this project is to demonstrate how arithmetic operations can be implemented in a mobile application. It helps beginners understand how user input, program logic, and user interface work together in Android development.

The application also includes a splash screen that appears when the app starts, giving the application a more professional look.

## Features
The calculator supports the following features:

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (%)
- mod(/)
- Simple and user-friendly interface
- Instant calculation results
- Splash screen displayed when the app launches


## Technologies Used
This project uses the following technologies:

- **Android Studio** – For developing and running the Android application
- **Java** – Programming language used for implementing the calculator logic
- **XML** – Used for designing the user interface
- **Android SDK** – Provides the tools and libraries required for Android development
